## 📖 Description

Nearby is a mobile application offering a benefit club with coupons to use at establishments near you.

## 🚀 Features

- ✅ View establishments near you.
- ✅ Filter establishments according to your needs (Shopping, food, accommodation, etc.).
- ✅ Scan the establishment's QRCode and enjoy your benefit.

## 🛠️ Technologies Used

The main technologies and tools used for the project development:

- [Kotlin](https://kotlinlang.org/)
- [Jetpack Compose](https://developer.android.com/develop/ui/compose/kotlin?hl=en)

## 📦 Installation and Usage

Follow these steps to install and run the project locally:

### Steps

1. Clone the repository:
   ```bash
   git clone https://github.com/marcelogaldino/nearby
   ```
2. Navigate to the project directory:
   ```bash
   cd nearby
   ```
3. Open the project with Android Studio and start coding.
